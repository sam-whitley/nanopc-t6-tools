## NanoPC-T6 DevTools

Here are some tools I used to flash the bootloader onto the SPI NOR Flash. 

- DriverAssistant (v.5.1.1) 
- MiniLoaderAll.bin
- RKDevTool (v3.15) 
- U-boot-friendlyelec-rk3588 (also compatible with the Orange Pi 5 Plus and likely other RK3588 SBCs)
  - idbloader.img
  - rkspi_loader.img
  - u-boot.itb
